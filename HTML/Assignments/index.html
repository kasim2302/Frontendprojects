HTML Intro

HTML stands for HyperText Markup Language. It defines the structure and content of a webpage. HTML elements are represented by tags enclosed in < >. Tags usually come in pairs: opening <tag> and closing </tag>. The latest version is HTML5. HTML works along with CSS (for styling) and JavaScript (for behavior).
Basic structure includes:
<!DOCTYPE html> – declares HTML5 version
<html> – root element
<head> – meta data and links
<body> – visible page content

Block-Level Elements

Block-level elements start on a new line and take the full width available. They create a block or section on the page.
Common block-level elements:
<div> – generic container
<p> – paragraph
<h1> to <h6> – headings
<section> , <article> , <nav> , <aside> – semantic blocks
<header> , <footer> – page or section headers/footers
<ul>, <ol>, <li> – lists
<table> – table structure

Inline-Level Elements

Inline elements do not start on a new line. They only take up as much width as necessary.
Common inline elements:
<span> – generic inline container
<a> – hyperlinks
<strong> – bold (important text)
<em> – italic (emphasized text)
<b> <i>, <ul> – stylistic text formatting
<img> – inline image
<label> – form label <small>, <sub> ,<sup> – smaller or positioned text

Semantic Elements

Semantic elements give meaning to the content. They help with SEO and accessibility.
Examples:
<header>–header section of page/section
<nav>–navigation links
<main>–main content area
<section>–a thematic grouping of content
<article>–independent piece of content (blog, news, etc.)
<aside>–sidebar or related info
<footer>– bottom section
<figure> & <figurecaption>– images with captions
