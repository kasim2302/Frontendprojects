Block Level Elements:

  • Starts on a new line by default (it “blocks” the next element from being beside it).
  • Takes up the full width available (even if the content inside is small).
  • Is often used to structure the layout of a web page.

Inline Level Elements:

  • Does not start on a new line(it flows within the text).
  • Takes up only as much width as its content needs.
  • Can only contain other inline elements or text (not block-level elements).

Why I chose these tags

I chose <section>, <div>, <h1>, <p>, and <ul> as block-level elements because they naturally start on a new line
and help structure a web page. For inline elements, I used <i>, <b>, <strong>, <em>, and <a> because they stay
within the flow of text and highlight or link specific words without breaking the line.
