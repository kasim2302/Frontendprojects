DOM (Document Object Model):

Definition:
DOM stands for Document Object Model. It represents the structure of an HTML document as a tree of objects that JavaScript can interact with.
It allows developers to dynamically access, modify, and update elements, styles, and content on a webpage.

ðŸ”¹ Key Points:

DOM connects HTML and JavaScript â€” allowing you to make webpages dynamic and interactive.

Without DOM, JavaScript cannot interact with webpage content (it would remain static).

DOM represents every HTML tag as an object (e.g., <p>, <div>, <button>).

You can add, modify, delete, and style elements using DOM methods.

It automatically updates the view when changes occur.

ðŸ”¹ Common DOM Selection Methods:

document.getElementById("id") â†’ Selects element by ID

document.getElementsByClassName("class") â†’ Selects elements by class

document.getElementsByTagName("tag") â†’ Selects elements by tag name

document.querySelector("selector") â†’ Selects first element matching CSS selector

document.querySelectorAll("selector") â†’ Selects all elements matching selector

ðŸ”¹ Example:
<h2 id="title">Welcome</h2>
<script>
    const heading = document.getElementById("title");
    heading.style.color = "blue";
    heading.innerText = "Hello, DOM!";
</script>

ðŸ§® 4. Array Methods in JavaScript

Arrays in JavaScript come with many built-in methods for processing and transforming data efficiently.

ðŸ”¹ reduce()

The reduce() method executes a function on each element to produce a single output value.

Use cases: sum, average, product, or combining data.

Syntax:

array.reduce((accumulator, currentValue) => {
  // logic
}, initialValue);


Example:

const numbers = [1, 2, 3, 4];
const total = numbers.reduce((acc, num) => acc + num, 0);
console.log(total); // 10

ðŸ”¹ map()

The map() method creates a new array by applying a function to every element.

Syntax:

array.map((currentValue, index, array) => expression);


Example:

const numbers = [1, 2, 3];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6]

ðŸ”¹ filter()

The filter() method creates a new array with elements that pass a given test.

Syntax:

array.filter(currentValue => condition);


Example:

const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // [2, 4]

ðŸ”¹ forEach()

The forEach() method executes a function once per array element.
It is mainly used for looping and performing side effects.

Syntax:

array.forEach((currentValue, index, array) => {
  // logic
});


Example:

const fruits = ["apple", "banana", "mango"];
fruits.forEach(fruit => console.log(fruit));

ðŸŒ 5. Fetch API

The fetch() function is used to get or send data between a client and a server using HTTP requests.
It works asynchronously and returns a Promise, allowing modern web apps to fetch data without reloading.

ðŸ”¹ Syntax:
fetch(url, options)
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));

ðŸ”¹ Example (GET Request):
fetch("https://api.example.com/data")
  .then(response => response.json())
  .then(data => console.log(data));

ðŸ”¹ Example (POST Request):

  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ name: "Kasim", age: 23 })
})
.then(res => res.json())
.then(data => console.log(data))
.catch(err => console.error(err));
