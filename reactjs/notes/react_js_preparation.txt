ðŸ”¹ Array

An array is a collection of elements stored in a single variable.
Example:

const fruits = ["Apple", "Banana", "Mango"];
console.log(fruits[1]); // Banana


Arrays are used to store lists of data like numbers, strings, or objects.

ðŸ”¹ Array of Objects

When each element of an array is an object, it is called an array of objects.
Example:

const students = [
  { name: "Kasim", age: 22 },
  { name: "Ali", age: 23 }
];
console.log(students[0].name); // Kasim

ðŸ”¹ Dot Notation

Dot notation is used to access object properties.
Example:

console.log(car.brand); // BMW


You can also use bracket notation like car["color"].

A callback function is a function passed as an argument to another function to be executed later.
Example:

  console.log("Hello " + name);

}

  console.log("Goodbye!");
